online presence, coding platforms, email etiquette, 
and cybercrime awareness.

## Module Summaries

| Task | Module | Description |
|------|--------|-------------|
| Task 1 | Digital Literacy Infographic | Created a yellow-themed infographic covering What is Digital Literacy, Safe Internet Practices, and Professional Online Presence using Canva |
| Task 2 | Student Digital Portfolio | Set up profiles on GitHub, LinkedIn, and Kaggle with professional information |
| Task 3 | Coding & Collaboration Platforms | Completed a beginner challenge on HackerRank and built a Digital Literacy Quiz using Google Forms |
| Task 4 | Email Etiquette | Drafted two professional emails and created a Social Media Do's and Don'ts checklist |
| Task 5 | Cybercrime Awareness | Written a real-life phishing case study and created a prevention checklist for college students |
